This is a car rental management system, built with Java + JavaFx and abit of css for styling. The system has features for adding vehicles (Cars, Trucks and Bikes), 
Renting vehicles, removing rentals, viewing rentals, adding customers, viewing available customers.
This was a school project. It uses file based database for storing data.
